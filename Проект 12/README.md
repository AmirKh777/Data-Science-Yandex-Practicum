# Описание проекта -  Магазину нужен инструмент, который будет искать токсичные комментарии и отправлять их на модерацию.  

Таблица, со следующими столбцами.
Столбец text в нём содержится текст комментария, а toxic — целевой признак.

## Задача
Обучите модель классифицировать комментарии на позитивные и негативные. В вашем распоряжении набор данных с разметкой о токсичности правок.
Постройте модель со значением метрики качества F1 не меньше 0.75. 

## Библиотеки

*numpy*

*TfidfVectorizer*

*re*

*nltk*

*stopwords*

*spacy*
